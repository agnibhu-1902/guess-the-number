# guess-the-number 
A command-line guessing game where the player needs to find the mystery number lying within 1 and 100.
### How to compile?
> Use 'gcc' or 'clang' to compile 'main.c'.
```
gcc -o main main.c
```
```
clang -o main main.c
```
### How to execute?
> Run the 'main' executable from the command-line.
#### Windows:
```
.\main.exe
```
#### MacOS (Unix):
```
./main
```
